# 微信公众号（服务号）接入chatGPT

> 首先需要注册open AI并生成key，参考教程[openAi](https://platform.openai.com/)

## 使用方法

**下载代码**

```bash
git clone https://github.com/bbxyczj/wx-chat.git
```

**运行服务**

```bash
java -jar wx-chat-0.1.1.jar 
```


## 特性

同步返回微信`success`,通过微信客服消息推送ChatGpt返回消息
